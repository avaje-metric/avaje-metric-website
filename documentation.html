Maven dependencies

The following dependencies are the API of avaje-metric-api and default implementation provided by avaje-metric-core

```xml org.avaje.metric avaje-metric-api 3.5.0 org.avaje.metric avaje-metric-core 3.5.0 ```

Maven plugin enhancement

Add the maven plugin below specifying the packages that should be scanned for classes annotated with @Singleton, @Path, @Service etc.

You can also use the Avaje Metrics @Timed annotation.

```xml org.avaje.metric enhance-maven-plugin 3.5.0 main process-classes target/classes org.example.myapp.** debug=1 enhance ```
